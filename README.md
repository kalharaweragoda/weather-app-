# 🌤️ Weather App - Real-Time Weather Forecast

A modern, responsive weather application that provides real-time weather data for any location worldwide. This project features a dynamic UI that changes background images based on the searched city.

---

## ✨ Features
* 🌍 **Real-Time Weather:** Get accurate weather details for any city in the world using the **Weather API**.
* 🖼️ **Dynamic Backgrounds:** High-quality city-specific images fetched dynamically via **Unsplash API**.
* 📱 **Responsive Design:** Optimized for a seamless experience across desktop, tablet, and mobile devices.
* ⚡ **Fast Performance:** Built with **Vite** for ultra-fast development and optimized production builds.

---

## 🛠️ Technologies Used
* **Frontend Library:** [React](https://reactjs.org/) (with TypeScript)
* **Language:** TypeScript & JavaScript
* **Styling:** HTML5 & CSS3
* **Build Tool:** [Vite](https://vitejs.dev/)
* **APIs:** * [OpenWeatherMap API](https://openweathermap.org/api) (or Weather API)
    * [Unsplash API](https://unsplash.com/developers)
